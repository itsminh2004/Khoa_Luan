ĐỒ ÁN TỐT NGHIỆP: Website quản lý và kinh doanh sản phẩm công nghệ tích hợp Chatbot AI
- Giới thiệu: Đây là đồ án tốt nghiệp với mục tiêu xây dựng một website quản lý và kinh doanh sản phẩm công nghệ, cho phép người dùng mua sắm trực tuyến và quản trị viên quản lý hệ thống một cách hiệu quả. Hệ thống được tích hợp Chatbot AI nhằm hỗ trợ người dùng tra cứu sản phẩm, tư vấn và giải đáp thắc mắc tự động.
- Mục tiêu đồ án: 
  + Vận dụng kiến thức đã học về lập trình Web và cơ sở dữ liệu.
  + Xây dựng hệ thống quản lý – bán hàng hoàn chỉnh.
  + Áp dụng mô hình MVC trong phát triển phần mềm.
  + Tích hợp Chatbot AI để nâng cao trải nghiệm người dùng.
  + Làm quen với quy trình phát triển, triển khai và quản lý dự án thực tế.
- Công nghệ sử dụng: 
  + Backend: Java, Spring Boot.
  + Frontend: HTML, CSS, JavaScript, JSP.
  + Cơ sở dữ liệu: MySQL.
  + ORM / JDBC: JPA / JDBC Template.
  + Chatbot AI: Mô hình RAG , Google gemini API.
  + Build Tool: Maven.
  + Server: Apache Tomcat.
- Chức năng chính:
  + Người dùng:
    + Đăng ký, đăng nhập tài khoản.
    + Xem danh sách sản phẩm công nghệ.
    + Tìm kiếm, lọc sản phẩm theo danh mục.
    + Xem chi tiết sản phẩm.
    + Thêm sản phẩm vào giỏ hàng.
    + Đặt hàng và theo dõi đơn hàng.  
    + Trò chuyện với Chatbot AI để được tư vấn sản phẩm.
  + Quản trị viên (Admin):
      + Quản lý danh mục sản phẩm (CRUD).
      + Quản lý sản phẩm (thêm, sửa, xóa, cập nhật tồn kho).
      + Quản lý người dùng.
      + Quản lý đơn hàng.
      + Thống kê doanh thu.
      + Quản lý nội dung chatbot (nếu có).
  + Chatbot AI: 
      + Hỗ trợ trả lời câu hỏi về sản phẩm.
      + Gợi ý sản phẩm phù hợp với nhu cầu người dùng.
      + Hỗ trợ 24/7, giảm tải cho quản trị viên.
